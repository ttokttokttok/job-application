# Screenshots Directory

This directory contains screenshots of the JobAgent application for use in the main README.

## Recommended Screenshot Names

To match the README, please name your screenshots as follows:

- `dashboard.png` - Main dashboard showing applications
- `resume-upload.png` - Resume upload interface
- `profile-form.png` - Profile management form
- `application-details.png` - Individual application details page
- `networking.png` - Networking dashboard
- `ai-agent.png` - AI agent interface

## Screenshot Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1200-1920px width
- **Quality**: High resolution for clarity
- **Content**: Show the most important features and UI elements

## Adding Screenshots

1. Place your screenshot files in this directory
2. Name them according to the list above
3. The main README will automatically display them

## Tips for Great Screenshots

- Use browser developer tools to capture clean screenshots
- Remove sensitive information before committing
- Consider using annotations or highlights for key features
- Ensure consistent styling across all screenshots
